```

├── index.html                              # 主页（Hero + Projects + In Progress + SHARDS + Achievements + Footer）
├── about.html                              # 关于页（人物志 + 能力矩阵 + 轨迹）
├── style.css                               # 全局样式 + 响应式（768/600/480px）
```

## 技术栈

- **HTML5 + CSS3 + Vanilla JS** — 零依赖
- **Google Fonts** — Fraunces / Noto Serif SC / JetBrains Mono / Special Elite
